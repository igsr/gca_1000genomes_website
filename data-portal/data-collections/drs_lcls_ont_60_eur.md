This collection contains long-read direct poly(A) RNA-seq data from 60 genetically diverse lymphoblastoid cell lines (LCLs) from the 1000 Genomes/Geuvadis project, generated using Oxford Nanopore Technologies.

Raw FAST5 files are archived in ENA under accession [PRJEB76585](https://www.ebi.ac.uk/ena/browser/view/PRJEB76585). These files can be browsed by sample, population, and data type below.

BAM files, transcript quantifications, m6A modification calls, and summary statistics are available from [Zenodo](https://doi.org/10.5281/zenodo.18763244).

Analysis code is available from the [DRS-LCLs-qtl-analysis GitHub repository](https://github.com/aline2593/DRS-LCLs-qtl-analysis).

Cell lines:

The data in this collection is from immortalised cell lines generated from samples collected by the 1000 Genomes Project. These cell lines are available from the Coriell institute.

Data reuse:

These data are released under the IGSR open data policy and are freely available for any use.
